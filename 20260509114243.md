# ローカルのドッカーの状態をWebUI で見る portainer

~/docker/portainer/docker-compose.yml に設定がされている

```
services:
  portainer:
    image: portainer/portainer-ce:latest
    container_name: portainer
    restart: unless-stopped

    ports:
      - "9000:9000"

    volumes:
      - /var/run/docker.sock:/var/run/docker.sock
      - portainer_data:/data

volumes:
  portainer_data:
```

restart の設定により、マシンが起動するといつでも実行される

## バージョンアップの方法

```
docker compose -f ~/docker/portainer/docker-compose.yml pull
docker compose -f ~/docker/portainer/docker-compose.yml up -d
```

すると、image に書いてある通り最新のイメージが pull されてそのイメージで起動しなおせる
