# C# コードフォーマッタ csharpier

ローカルのプロジェクトで csharpier をフォーマッタとして使用するには

1. dotnet new tool-manifest を実行してプロジェクトルートに dotnet-tools.json を作成する
2. dotnet tool install csharpier を実行して dotnet-tools.json にツールを登録する

csharpier は ~/.nuget/packages あたりにインストールされ使いまわされるが、グローバルオプションを付けなければプロジェクト毎に使う・使わない、を調整できる

3. dotnet csharpier format . でファイルをフォーマットして変更する --check オプションを付けるとチェックだけする
4. Rider IDE でも csharpier のプラグインがあるのでこれをインストール
   - Tools > CSharpier の設定から Run on save を設定する。
   - 逆に Actions on Save の自動フォーマットの設定はオフにする
