# gitignore ファイルを作成するショートカット gi

fish の関数として gi を作成している。

```fish
function gi
    if test (count $argv) -eq 0
        echo "usage: gi <preset> [extra args...]"
        return 1
    end
    
    set preset $argv[1]
    set rest $argv[2..-1]
    set file ~/.config/gitignore/presets/$preset.txt
    
    if not test -f $file
        echo "preset not found: $preset ($file)"
        return 1
    end
    
    gitnr create (cat $file) $rest
end
```

~/.config/gitignore/presets/ に dotnet.txt や web.txt といったファイルを作成して

```
tt:csharp
tt:dotnetcore
```

といった内容を入力しておく gi 関数は内部で gitnr コマンドを呼び出して gitignore ファイルを出力する
